Guess The Number

Guess The Number, a simple game using Local Seacrh Algorithm. The game developed is an interactive and entertaining number guessing game, uses a Tkinter-based graphical user interface (GUI).

Once the player selects a category, the game will start and display the number range which must be guessed. Players can enter their guessed numbers through a column input. Every time the player enters a guess, the system will provide clear feedback regarding this guess. If the player fails to guess the number, the system will automatically Minimize the range of numbers that must be guessed. The range of numbers given can be smaller or greater than the number that has been guessed by the player (depending on the number entered by the player). If players manage to guess the correct number, they will be congratulated and given information regarding the number of guesses needed to reach the correct number.

One of the interesting features of this game is the local search system implemented for helps players narrow down their possible guesses. With every guess done, the range of valid numbers for the next guess will be adjusted based on feedback is provided, thereby giving players better opportunities to guess faster. This adds a strategic dimension to the game, encouraging players to think more critically about their number choices.
